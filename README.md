Tic-Tac-Toe
===========

The classical game of Tic-Tac-Toe written in **TypeScript**.

Concepts used:

- classes
- encapsulation
- static typing
- modules
- callbacks
- importing modules

Installation
------------
- Install Node.js and npm
- Build `grunt`
- Start the game `node js/main.js`
